Intro

Hi there. My name is David, a data analyst who uses magic to extract insights from data. I have worked in consulting, education, and healthcare industries for the past 5 years automating the data processing, building alghoritms for targeting/goalsetting, creating/designing dashboards for presenting the results to C-Level Executives. I am highly specialized at collaborating SQL, Python, and Tableau while transforming tedious work into useful insights. Please check some of my awesome work.

My friends describe me as easy-going, creative, a music guru, gymrat and great cook. My favorite weekend activies include swimming, camping, and art exhibitions - Baroque, Modern, and TechArt are my favorites. Also, I am fluent in English, German, Azerbaijani, and Turkish which has helped me travel from Asia to Europe, to America.

Work

If I have to describe my work in a single sentence, I would put it this way: I use my technical skills and industry specific domain knowledge to extract insights from data and effectively communicate them to a non-technical audiance in an easily understandable language. Examples of my work are below:

SQL

Covid hit the world unexpectedly and so did the world of data. Thanks to Covid, data scientists received a large volume of data to analyze and find insights. I did a quick exploratory analysis using SQL on Covid data shared by the "Our World in Data" to find how covid was managed globally .

Please feel free to visit my github reposotory to find the queries that helped me in my analysis @SQL.

Python

There are over 1 million physicians, health care providers in the United States. Knowing which doctors should be targeted can make the work of the companies a lot easier. I have developed a targeting algorithm that finds the doctors prescribing specific products and ranks them based on their performance. This would allow companies to know what doctors to reach out without wasting their time.

Please feel free to visit my github reposotory to find the python code that helped me in my analysis @Python.

Tableau

The dashboard above has been designed and created in Tableau connected with SQL database. It is an interactive dashboard set to automate itself weekly without any manual interruption. The dashboard can be used for various reasons such as tracking the performance of the sales force and predicting their payout. The filters attached to the report allows the user easily to interract with the data without any supervision.

Please feel free to visit my tableau public account to see the report in detail @Performance.

The dashboard above has been designed and created in Tableau connecteed with SQL database. It is a highly interractive, user-friendly dashboard that helps the viewers to discover how much each employee has been paid. The filters help to find the employees with highest/lowest bonus pay.

Please feel free to visit my tableau public account to see the report in detail @Bonus.

Powerpoint

When time is limited, deadline is approacing, and insights from data have to be presented quickly, powerpoint can be one of the best alternatives. Its "easy to use and understand" nature allows to create impressive visualizations with data in few hours. Above is an example of salary comparison among various departments within organization.

Please feel free to visit my github reposotory to find the powerpoint presentation for making the above analysis @PowerPoint.
